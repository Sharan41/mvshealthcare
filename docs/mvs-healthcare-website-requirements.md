# MVS Healthcare Website - Requirements Document

## 1. Project Overview

### 1.1 Project Name
**MVS Healthcare** - Medical Disposables Catalogue Website

### 1.2 Project Type
Modern single-page application (SPA) catalogue website

### 1.3 Primary Purpose
- Showcase MVS Healthcare's medical disposable products
- Provide company information (Mission, Vision, Values)
- Serve as a digital catalogue for healthcare providers, distributors, and partners
- Generate inquiries and leads for product sales

### 1.4 Target Audience
- **Primary**: Healthcare providers (hospitals, clinics, nursing homes)
- **Secondary**: Medical distributors and wholesalers
- **Tertiary**: Healthcare procurement professionals
- **Geographic**: India and international markets (28+ countries based on competitor analysis)
